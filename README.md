# Todious

An Todo App built using Wails 3 + Nuxt 4. This app is example of [Wails3 template for Nuxt 4](https://github.com/seinkytarlicht/wails3-template-nuxt4-ts)

## Tech Stack

| Module   | Technology |
| -------- | ---------- |
| Desktop  | Wails 3    |
| Frontend | Nuxt 4     |
| UI       | Nuxt UI    |
| Database | gorm       |

## Development

Start the development server with hot-reload:

```bash
wails3 task dev
```

## Build

Compile and build app:

```bash
wails3 task build
```

For Production:

```bash
wails3 task package
```

The output binary will be available in the `bin/` directory.
